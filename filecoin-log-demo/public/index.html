How It Works

This demo app automatically captures all HTTP requests and sends them as logs to the Filecoin storage API.

Every request to any endpoint in this app generates a log entry with:

  • Timestamp
  • HTTP method (GET, POST, etc.)
  • Requested URL
  • Request headers
  • User agent
  • IP address
  • Response status code
  • Response time

These logs are then sent to the Filecoin storage API and stored on the Filecoin network.
